# Task-Manager
Projet web de gestion de tâches avec mongo et express
Vous pouvez :
  - Ajoutez vos tâches
  - Modifiez vos tâches
  - Recherchez vos tâches avec des mots-clés ou des filtres
  - Trier vos tâches

## Installer les dépendances :
```npm install```

## Créer le .env à la racine du projet :
```MONGO_URI:user```

## Lancer le serveur :  
```npm run start```

Vous pouvez maintenant ouvrir le site web (index.html) avec votre navigateurs préféré !
